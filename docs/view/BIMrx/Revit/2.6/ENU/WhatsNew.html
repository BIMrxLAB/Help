What's New

To download the latest Public Release of BIMrx for Revit:

https://www.microdesk.com/bimrx-download/


The v2.6 release of BIMrx® for Revit highlights the following features:

  • General
    • Units based on project settings for both Imperial and Metric across all UIs
    • Fatal Error no longer an issue in Family Editor mode
    • Hide Non-Activated tabs/vertical in Ribbon
    • Additional Tooltip videos added
  • BIMrx Core 
    • Property Parameter Manager feature
      • [Element][Location] - X, Y, Z location added (now works on design and fabrication parts)
    • Tag All 
      • Works with Rooms and Spaces 
    • Sheet View Manager 
      • View creation maintains renaming even when duplicates exist 
        • Any duplicates maintain the new naming and adds (#) to the end
  • BIMrx MEP  
    • Custom Elbow (Previously called Master Elbow) 
    • Elbow Up, Down, Left, Right  
      • Maintains all slopes 
    • Reroute Under/Over command 
      • Maintains Insulation on pipe and duct 
      • Maintains Lining on duct 
    • Dialog boxes
      • UI updated UI for all MEP dialog boxes
  • BIMrx Fabrication 
    • Hanger Array
      • UI updated for Hanger Array dialog box
    • Spool Manager
      • UI updated for Spool Manager dialog box
