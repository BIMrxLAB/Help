Starting the Command

In Revit:

  • User must be inside of a Revit schedule to activate the command.
  • The Parameters being pushed must be in the Shared Parameter file being used in the project.
  • In the ribbon, navigate to the BIMrx tab >> BIMrx MEP panel
  • Click the Parameter Pusher command to activate.


  • The Parameter Pusher dialog pops up. The user can now modify how the parameters are pushed into families:

 

  • The check boxes in the Add column will determine which parameters are pushed or not.
  • A checked box in the Inst column will make the parameters that are pushed come in as Instance parameters, an unchecked box means the parameters will be Type parameters.
  • The drop-down Group will allow the user to select which group the parameter will show up in.
  • The check boxes in the Mod column will determine which families the parameters are pushed to
  • The Save Families? check box allows the user to automatically save the families when complete.
    • With Save Families? selected, the user has the choice to Save to desktop or Save over original location.

  • Click Accept to push the families into the project.


Notes:

  • Upon Accepting Parameter Pusher settings, you will be required to overwrite the existing families in the project.
