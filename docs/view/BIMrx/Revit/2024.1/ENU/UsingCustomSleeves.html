Using Custom Sleeves

As noted in the Sleeves page, NAVIATE uses specific sleeve Revit Families that are configured to be used with the Fabrication Sleeves tool.


Follow the steps/guidelines below in order to use user custom Sleeve Families.


  • Within Revit, navigate to "C:\ProgramData\Microdesk\BIMrx Families\Imperial". The NAVIATE Sleeve Families are located at this directory.
  • Identify the .rfa Family file to edit and open in Revit. The bullet points below describe the case in which each Family is used.


Tip: Copy and paste these .rfa files to another location if needed to revert to original settings.


    • Duct Sleeves (Category: Duct Accessories)
      • BIMrx_DuctAccessory_Sleeve_Rectangular - placed when the Wall Sleeves, Floor/Roof Sleeves, or BlockOut Sleeves  command is ran for Rectangular Ducts.
      • BIMrx_DuctAccessory_Sleeve_Rectangular_Single - placed when the Manual Sleeves command is ran for Rectangular Ducts.
      • BIMrx_DuctAccessory_Sleeve_Round - placed when the Wall Sleeves, Floor/Roof Sleeves, or BlockOut Sleeves  command is ran for Round Ducts.
      • BIMrx_DuctAccessory_Sleeve_Round_Single - placed when the Manual Sleeves command is ran for Round Ducts.
    • Pipe Sleeves (Category: Pipe Accessories)
      • BIMrx_PipeAccessory_Sleeve_Rectangular - placed when the BlockOut Sleeves command is ran for Pipes.
      • BIMrx_PipeAccessory_Sleeve_Rectangular_Single - placed when the BlockOut Sleeves command is ran for Pipes.
      • BIMrx_PipeAccessory_Sleeve_Round - placed when the Wall Sleeves or Floor/Roof Sleeves command is ran for Pipes.
      • BIMrx_PipeAccessory_Sleeve_Round_Single - placed when the Manual Sleeves command is ran for Pipes.
  • Parameters - Rectangular Sleeves
    • The Family contains three (3) important parameters that drive its dimensions: (1) BIMrx_SLEEVE WIDTH, (2) BIMrx_SLEEVE HEIGHT, (3) BIMrx_SLEEVE DEPTH. Ensure that these dimension parameters are maintained and control the essential geometry forms. 

  • Parameters - Round Sleeves
    • Similar to Rectangular Sleeves, this Family contains the same three (3) important parameters that drive its dimensions: (1) BIMrx_SLEEVE WIDTH, (2) BIMrx_SLEEVE HEIGHT, (3) BIMrx_SLEEVE DEPTH. 
    • For Round Sleeves, (1) BIMrx_SLEEVE WIDTH and (2) BIMrx_SLEEVE HEIGHT are set equal to each other via Formulas in Family Type Parameters. Ensure that these dimension parameters are maintained and control the essential geometry forms.

  • Overwrite the 'BIMrx...' sleeve Families with the user custom sleeve Families.
  • Load updated Families to Project
    • If the 'BIMrx...' sleeve Families already exist in the current Revit model, delete them from the model by navigating to the Project Browser >> Families >> Duct/Pipe Accessories, right-clicking the Families, and choosing Delete.
    • NAVIATE Fabrication tab >> Fab Model panel >> Sleeve Options drop-down button list.
      • This button will load in the Families from the directory "C:\ProgramData\Microdesk\BIMrx Families\Imperial".
  • Size Mapping
    • NAVIATE Fabrication tab >> Fab Model panel >> Sleeve Options drop-down button list.
    • Round Sleeves
      • The Sleeve Size column associates with the Family Type Parameter "BIMrx_SLEEVE WIDTH".

    • Rectangular Sleeves
      • The dimensions (Width and Height) of Rectangular Sleeves directly associate with the Famiyl Type Parameters (1) BIMrx_SLEEVE WIDTH and (2) BIMrx_SLEEVE HEIGHT. Refer to Size Mapping for information on how to add minimum gap space around the rectangular host element (see below).
