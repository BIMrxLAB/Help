New project members can be added to one or more BIM 360Â® projects. They can be added when Multi-Select is set to either On or Off. If Company is set to blank, then the system will use the default company assigned to the member. If no default is assigned, it will be left blank.

Likewise, one or more project Roles can be assigned. As with the company, if no Roles are assigned the system will add a default Role.
